# Smart-CPU-Cooling-System-Using-Arduino

### Overview
This project presents an **intelligent CPU cooling system** that dynamically adjusts fan speed based on real-time temperature readings using an **Arduino Uno** microcontroller. Designed to prevent overheating, thermal throttling, and extend hardware lifespan, the system provides an efficient, responsive, and cost-effective cooling solution for desktop computers.

### Key Highlights
- **Microcontroller**: Arduino Uno Rev3
- **Temperature Sensor**: DHT11 (Temperature & Humidity)
- **Control**: Relay module to switch 12V CPU cooling fan
- **Display**: I2C LCD for real-time temperature and humidity monitoring
- **Logic**: Threshold-based fan control (fan turns ON when temperature exceeds set limit)
- **Features**: Adaptive cooling, energy efficiency, reduced noise

### Objectives
- Monitor CPU temperature in real-time using sensors
- Automatically control cooling fan speed via Arduino
- Optimize energy consumption and reduce unnecessary fan operation
- Display live temperature and system status on LCD
- Provide hands-on experience in embedded systems and sensor interfacing

### Results
- Successfully maintains CPU temperature within safe limits under varying loads
- Effective dynamic response with fan activation above temperature threshold
- Real-time monitoring and display of temperature & humidity
- Demonstrated reliable performance and energy-efficient operation

### Technologies & Components
- **Arduino Uno + Arduino IDE**
- DHT11 Temperature & Humidity Sensor
- 12V DC Fan + Relay Module
- I2C LCD Display
- Embedded C Programming
